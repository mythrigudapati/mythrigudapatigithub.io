Bit: A binary digit, either 0 or 1.

Byte: A sequence of 8 bits.

Roundoff: Error that results when the number of bits is not enough to represent the number with full precision.

Analog Data: Values that change smoothly, rather than in discrete intervals, over time (e.g., pitch or volume of music).

Lossless: Compression that preserves all original data and allows full recovery.

Lossy: Compression that discards some data and cannot fully recover the original.

Metadata: Data about data, such as information about a file or a record.

Computing Device: A physical device that can run a program (like a phone or computer).

Computer Network: A group of connected devices that can send or receive data.

Path: The series of connections data travels through in a network.

Protocol: A set of rules for how devices communicate.

Fault Tolerant: A system that still works even if some parts fail.

Scalability: A system’s ability to grow or shrink as needed.

Bandwidth: The max amount of data that can be sent in a given time (measured in bits per second).

URL: A web address, like www.example.com.

IP: Assigns addresses to devices and routes data.

TCP: Ensures reliable delivery of data.

UDP: A faster, less reliable way to send data.

Packets: Small chunks of data used in internet communication.

Routers: Devices that forward data and choose the best path.

DNS: Translates domain names into IP addresses.

HTTP: Protocol used to load web pages.

WWW: The system of linked documents and media on the internet.

Redundancy: Multiple paths exist in a network for reliability.

Scalability: The internet can grow without major changes.

IPv4: Uses 32-bit addresses for ~4.3 billion unique devices.

IPv6: Adds 16 more bits, allowing 65,536x more addresses.

E-Waste & the Internet’s Environmental Impact

The internet requires massive data centers and servers that consume huge amounts of energy, contributing to carbon emissions. Electronic waste (E-Waste) from outdated devices creates environmental hazards if not recycled properly. Many companies and organizations are working on improving energy efficiency and recycling methods to reduce this impact.

Fun Fact: If the internet were a country, it would rank as one of the world's largest consumers of electricity!
